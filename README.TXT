# Projeto: Sobrevivendo (e Aprendendo) Java

## Objetivo do Projeto

Registrar minha jornada de aprendizado em Java, um exercício de cada vez. O objetivo principal é domar este monstro cheio de `{` chaves `}` e pontos-e-vírgulas, e talvez, só talvez, começar a gostar dele (quem sabe, né). Cada pasta representa um pequeno passo na minha luta para entender o que é um `public static void main(String[] args)`.

### Como Iniciar o Projeto

1.  **Pré-requisitos:** Uma boa dose de paciência e um monster gelado. Ah, e ter o [Java Development Kit (JDK)].
2.  **Execução:**
    * Navegue até a pasta de um dos exercícios.
    * Compile todos os arquivos `.java` com o comando: `javac *.java`
    * Execute o arquivo principal (aquele que contém o `main`) com o comando: `java NomeDaClassePrincipal`
    * Reze para a JVM (Java Virtual Machine) ser piedosa e tudo funcionar como esperado (risos nervosos).
