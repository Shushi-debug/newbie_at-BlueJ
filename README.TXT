Project: Surviving (and Learning) Java

Project Objective
To document my Java learning journey, one exercise at a time. The main goal is to tame this monster full of { curly braces } and semicolons, and maybe, just maybe, start to like it (who knows, right?). Each folder represents a small step in my struggle to understand what a public static void main(String[] args) actually is.

Author:
Stefani, aka Shushi (in that order) - The brave (and slightly terrified) developer facing down Java's syntax.

Instructions for the User (or for my future self... hello there!)
If you came here expecting super-optimized code from a Java Jedi master, you might be in the wrong repository. But if you're looking for the honest record of someone who is learning, stumbling, and eventually getting it right, then make yourself at home!

To Run: Follow the steps under "How to Get Started".

To Understand the Suffering: Read the comments in the code (if I remember to add them).

To Help: If you find a bug or a simpler way to do something, feel free to open an "Issue". All help in taming this beast is welcome!

>>How to Get Started!<<
Prerequisites: A healthy dose of patience and a cold Monster energy drink. Oh, and you'll also need the [Java Development Kit (JDK)].

Execution:
-Navigate to the folder of one of the exercises.
-Compile all .java files with the command: javac *.java
-Run the main file (the one containing the main method) with the command: java MainClassName
-Pray that the JVM (Java Virtual Machine) is merciful and that everything works as expected (nervous laughter).



Projeto: Sobrevivendo (e Aprendendo) Java

Objetivo do Projeto

Registrar minha jornada de aprendizado em Java, um exercício de cada vez. O objetivo principal é domar este monstro cheio de `{` chaves `}` e pontos-e-vírgulas, e talvez, só talvez, começar a gostar dele (quem sabe, né). Cada pasta representa um pequeno passo na minha luta para entender o que é um `public static void main(String[] args)`.

Autora:
Stefani, vulgo Shushi (é essa a ordem?) - A corajosa (e levemente apavorada) desenvolvedora enfrentando a sintaxe de Java.

Instruções para o Usuário (ou para o meu "eu" do futuro... olá!)?

Se você veio aqui esperando um código super otimizado de um mestre Jedi do Java, talvez tenha entrado no repositório errado. Mas se você está procurando o registro honesto de alguém que está aprendendo, tropeçando e eventualmente acertando, sinta-se em casa!

Para Executar: Siga os passos acima em "Como Iniciar o Projeto".
Para Entender o Sofrimento: Leia os comentários no código (se eu me lembrar de colocá-los).
Para Ajudar: Se encontrar um erro ou uma forma mais simples de fazer algo, sinta-se à vontade para abrir uma "Issue". Toda ajuda para domar essa fera é bem-vinda!

>>Como Iniciar o Projeto:<<

Pré-requisitos: Uma boa dose de paciência e um monster gelado. Ah, e ter o [Java Development Kit (JDK)].
Execução:
-Navegue até a pasta de um dos exercícios.
-Compile todos os arquivos `.java` com o comando: `javac *.java`
-Execute o arquivo principal (aquele que contém o `main`) com o comando: `java NomeDaClassePrincipal`
-Reze para a JVM (Java Virtual Machine) ser piedosa e tudo funcionar como esperado (risos nervosos).
